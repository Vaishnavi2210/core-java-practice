# core-java-practice
Core Java programs topic-wise

This repository contains Core Java programs organized topic-wise for practice and revision.

## Purpose
- To strengthen Core Java fundamentals
- To practice Java concepts with hands-on examples
- To maintain a structured reference for interview preparation

## Topics Covered
- Basics (Variables, Data Types, Operators)
- Conditional Statements (if-else, switch)
- Loops (for, while, do-while)
- Patterns (Star and Number Patterns)
- Arrays
- Strings
- Class and Object
- Constructors
- Inheritance
- Polymorphism
- Abstraction (Abstract Class & Interface)
- Exception Handling

## Structure
Each topic is placed in a separate folder and contains simple, easy-to-understand Java programs with proper naming and comments.

## Technologies Used
- Java (Core Java)
- Git & GitHub

## How to Run
1. Clone the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run individual `.java` files
